Masshelper - Your new secret to success

Introduction

Masshelper is the new, and best way to look up how to massproduce a certain splicable item! Tons of work has went into this, and we're looking to expand our database as soon as new updates come out!



Please note that Possibilities Mode is a standalone option and won't work with other options! Also worth noting that Classic Style would be best fitting for handheld devices. If you want to save yourself some time and potential world locks, you can try using Compact Mode.

DEPRECATION WARNING: Masshelper will be officialy shut down May 8th, 2018.

If you still depend on masshelper, don't worry, I've released the source code of it to GitHub
